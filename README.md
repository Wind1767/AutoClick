# Auto Clicker
- A lightweight utility to automate mouse clicking tasks.

## Features
* Customizable Intervals: Set intervals in hours, minutes, seconds, and milliseconds.
* Mouse Options: Supports Left, Right, and Middle clicks (Single/Double).
* Customize Hotkey: Start/Stop instantly with F8 (Customizable).

## Usage
1. Open AutoClicker.exe.
2. Set your clicking interval and preferred mouse button.
3. Press F8 to Start/Stop.
